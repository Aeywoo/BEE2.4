Installation instructions:
1. Unzip to any desired location.
2. Merge the packages zip (https://github.com/TeamSpen210/BEE2-items/releases) with the extracted folders.
3. Run the BEE2 executable.
4. To run the backup tool standalone, run 'backup_tool.exe' instead.
5. You may also run compiler_options.exe, to edit compiler options while the game is running.
